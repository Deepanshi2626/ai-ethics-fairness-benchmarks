# AI Ethics Fairness Benchmarks

Supplementary code and experiments for the paper *"From Principles to Practice: Measuring AI Ethics Compliance Using Fairness Benchmarks"* (Deepanshi & Dr. Chhavi Rana, UIET, MDU Rohtak).

## Overview

This repository contains the experimental pipeline used to benchmark fairness across three datasets (UCI Adult Income, ProPublica COMPAS, German Credit) using three classifiers (Logistic Regression, Random Forest, XGBoost), with three bias mitigation strategies (Reweighing, Adversarial Debiasing, Equalized Odds Post-Processing).

## Datasets

- **UCI Adult Income** — downloaded directly from the UCI ML Repository (48,842 raw records; 45,222 after cleaning)
- **ProPublica COMPAS** — downloaded from ProPublica's public GitHub repository, filtered using ProPublica's standard criteria (5,278 final records, Black/White defendants only)
- **German Credit** — downloaded from the UCI ML Repository (1,000 records)

## Methods

- Baseline models evaluated with 5-fold stratified cross-validation
- Fairness metrics computed: Disparate Impact (DI), Demographic Parity Difference (DPD), Equalized Odds Difference (EOD), Average Odds Difference (AOD)
- Mitigation methods: Reweighing (Kamiran & Calders), Adversarial Debiasing (PyTorch implementation), Equalized Odds Post-Processing (grid threshold search)

## Reproducibility

Open `fairness_benchmarks.ipynb` in Google Colab and run all cells in order. Library versions used are printed in the second cell of the notebook.

## Note on results

All results in this repository were empirically generated by running the code in this notebook. No results were fabricated or estimated.
