# AI Ethics Fairness Benchmarks

Supplementary code and experiments for the paper *"From Principles to Practice: Measuring AI Ethics Compliance Using Fairness Benchmarks"* (Deepanshi & Dr. Chhavi Rana, UIET, Maharshi Dayanand University, Rohtak).

## Overview

This repository contains the complete experimental pipeline used to benchmark fairness across three datasets using three classifiers, three bias mitigation strategies, and two explainability methods.

## Datasets

- **UCI Adult Income** — downloaded directly from the UCI ML Repository (48,842 raw records combining `adult.data` and `adult.test`; 45,222 after removing missing values)
- **ProPublica COMPAS** — downloaded from ProPublica's public GitHub repository, filtered using ProPublica's standard screening criteria (5,278 final records, African-American and Caucasian defendants only)
- **German Credit** — downloaded from the UCI ML Repository (1,000 records)

## Methods

- **Classifiers:** Logistic Regression, Random Forest, XGBoost
- **Baseline evaluation:** 5-fold stratified cross-validation, with mean and standard deviation reported for all metrics
- **Fairness metrics:** Disparate Impact (DI), Demographic Parity Difference (DPD), Equalized Odds Difference (EOD), Average Odds Difference (AOD)
- **Mitigation methods:**
  - Reweighing (Kamiran & Calders pre-processing method)
  - Adversarial Debiasing (custom PyTorch implementation, classifier + adversary trained jointly)
  - Equalized Odds Post-Processing (grid threshold search balancing fairness gap against accuracy)
- **Explainability:**
  - SHAP (TreeExplainer) — global feature importance and subgroup-level analysis across all three datasets
  - LIME (LimeTabularExplainer) — instance-level analysis on Adult Income (female low-income predictions) and COMPAS (false-positive predictions by race)

## Reproducibility

Open `ai_ethics_fairness_benchmarks_complete.ipynb` in Google Colab and run all cells in order (Runtime → Run all). Library versions used are printed where relevant; the pipeline was last verified against Python 3.12, scikit-learn 1.6.1, XGBoost 3.3.0, fairlearn 0.14.0, and shap 0.52.0.

## Note on results

All results in this repository were empirically generated by running the code in this notebook. No results were fabricated, estimated, or extrapolated. Numbers reported in the associated manuscript match the outputs of this notebook.
